This project is created to addresses Task 1 and T 2 of the 2020 Ashesi Career Fair Programming Challenge.
The results from Task 1 can be obtained from running "Answer.java". This produces a file called "task1_solution-covid_data.txt" 
The results from Task 2 can be obtained from running "SecondTaskMain.java". This produces a file "task2_solution-partial_time_series_file.txt"
